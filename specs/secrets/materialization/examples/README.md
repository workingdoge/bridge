# Materialization Examples

Canonical examples live under [`../../secret-0002/examples/`](../../secret-0002/examples/).

Key examples:

- backend, materializer, and binding examples for analytics database and
  signing-key paths;
- plan requests for accepted, burn, and signing-key materialization paths;
- generated materialization sessions for allow and deny outcomes;
- signer request, response, and witness-realization examples.

Use these as fixture-level examples of the `SECRET-0002` realization contract,
not as live provider bindings.
